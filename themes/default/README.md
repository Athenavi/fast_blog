# FastBlog Default Theme

FastBlog 默认主题 - 简洁、现代、响应式设计

## 特性

- ✅ 响应式设计,支持移动端
- ✅ 深色模式支持
- ✅ 侧边栏可配置(左/右/隐藏)
- ✅ 自定义颜色方案
- ✅ 灵活的排版设置
- ✅ 评论系统集成
- ✅ 社交分享按钮
- ✅ 相关文章推荐
- ✅ 目录导航

## 配置

在 `metadata.json` 中修改主题配置:

```json
{
  "colors": {
    "primary": "#3b82f6",
    "secondary": "#64748b"
  },
  "layout": {
    "sidebar_position": "right",
    "content_width": "max-w-4xl"
  }
}
```

## 自定义

1. 复制 `default` 主题目录
2. 修改 `metadata.json` 中的主题信息
3. 调整 `theme.config.js` 中的配置
4. 在后台激活新主题

## 支持的功能

- Custom Logo
- Custom Header
- Featured Image
- Post Thumbnails
- Comments
- Widgets
