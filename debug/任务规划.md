## 三、插件系统详解

### 3.1 EventBus 事件

| 事件名 | 类型 | 触发位置 | 已订阅者 |
|--------|------|---------|---------|
| article.published | emit | articles.py | newsletter -> 自动推送邮件 |
| article.updated | emit | articles.py | - |
| article.deleted | emit | articles.py | - |
| article.content | pipeline | articles.py GET detail | code-snippets -> [snippet:N] 嵌入 |
| comment.created | emit | comments.py | - |
| comment.content | pipeline | comments.py | katex-render -> LaTeX 渲染 |
| theme.activated | emit | 主题系统 | 3 个主题插件 -> CSS 切换 |

### 3.2 当前插件

| 插件 | 类别 | 数据存储 | EventBus | 前端管理页 |
|------|------|---------|----------|-----------|
| code-snippets | content | SQLite | article.content 管道 | /admin/plugin-pages/code-snippets |
| newsletter | content | SQLite | article.published 事件 | /admin/plugin-pages/newsletter |
| article-likes | social | SQLite | - | /admin/plugin-pages/article-likes |
| katex-render | content | 无 | comment.content 管道 | /admin/plugin-pages/katex-render |
| popular-articles | widget | 无 | - | /admin/plugin-pages/popular-articles |
| approval | tool | 无 | - | /admin/plugin-pages/approval |
| enterprise | tool | 无 | - | /admin/plugin-pages/enterprise |
| migration | tool | 无 | - | /admin/plugin-pages/migration |
| fastblog-default | theme | 无 | theme.activated | /admin/plugin-pages/fastblog-default |
| magazine | theme | 无 | theme.activated | /admin/plugin-pages/magazine |
| modern-minimal | theme | 无 | theme.activated | /admin/plugin-pages/modern-minimal |

### 3.3 新增插件流程

```
plugins/<slug>/
  metadata.json              # 插件声明
  plugin.py                  # BasePlugin 子类 + subscribers()
  frontend/                  # 前端（可选，自动扫描）
    manifest.json            # 路由 + 侧边栏声明
    admin/Page.tsx           # 管理页面组件
    api.ts                   # API 服务（调 action 端点）
```

npm run prescan（内含在 build/dev 中）自动发现并生成代理页面。

|------|----------|-----------|------|------------|
| 架构 | FastAPI + Astro (异步) | PHP + MySQL (同步) | Java + Spring | Go + React |
| API 风格 | REST + MCP (AI) | REST | REST + GraphQL | GraphQL |
| 插件数量 | 11 | 60,000+ | 100+ | 插件市场 |
| 主题数量 | 3 | 10,000+ | 50+ | - |
| 移动端 | PWA + Capacitor 脚手架 | iOS/Android App | 移动端适配 | - |

| 社区规模 | 单人项目 | 全球最大 CMS 社区 | 中国活跃社区 | 海外社区 |
| 开源协议 | Apache 2.0 | GPLv2 | GPLv3 | 专有/开源版 |
