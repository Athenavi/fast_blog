## 三、插件系统详解

### 3.1 EventBus 事件

| 事件名 | 类型 | 触发位置 | 已订阅者 |
|--------|------|---------|---------|
| `article.published` | emit | `articles.py` POST `/` | **newsletter** → 自动推送邮件 |
| `article.updated` | emit | `articles.py` PUT `/{id}` | — |
| `article.deleted` | emit | `articles.py` DELETE `/{id}` | — |
| `article.content` | pipeline | `articles.py` GET detail | **code-snippets** → `[snippet:N]` 嵌入 |
| `comment.created` | emit | `comments.py` POST | — |

### 3.2 当前插件

| 插件 | 后端 | 数据存储 | EventBus | 前端管理页 |
|------|------|---------|----------|-----------|
| **code-snippets** | Python CRUD | `storage/plugins/code-snippets/data.db` | `article.content` 管道 | `/admin/plugin-pages/code-snippets` |
| **newsletter** | Python CRUD + SMTP | `storage/plugins/newsletter/data.db` | `article.published` 事件 | `/admin/plugin-pages/newsletter` |
| **default** (主题) | ThemePlugin | 无（CSS+配置） | `theme.activated` | `/admin/plugin-pages/default` |
| **magazine** (主题) | ThemePlugin | 无（CSS+配置） | `theme.activated` | `/admin/plugin-pages/magazine` |
| **modern-minimal** (主题) | ThemePlugin | 无（CSS+配置） | `theme.activated` | `/admin/plugin-pages/modern-minimal` |

### 3.3 新增插件流程

```bash
plugins/<slug>/
├── metadata.json              # 插件声明
├── plugin.py                  # BasePlugin 子类 + subscribers()
└── frontend/                  # 前端（可选，自动扫描）
    ├── manifest.json          # 路由 + 侧边栏声明
    ├── admin/Page.tsx         # 管理页面组件
    └── api.ts                 # API 服务（调 action 端点）
```

`npm run prescan`（内含在 `build`/`dev` 中）自动发现并生成代理页面。
