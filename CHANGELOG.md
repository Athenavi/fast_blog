# Changelog

All notable changes to FastBlog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Bilingual documentation (English & Chinese)
- Comprehensive CI/CD pipeline
- Docker Compose production configuration
- Makefile for common development tasks
- Security policy documentation
- Root-level contributing guide
- EditorConfig for consistent formatting

### Changed

- Improved project documentation structure
- Enhanced README with better organization

## [0.1.0] - 2025-01-01

### Added

- FastAPI backend with async support
- Django backend option
- Astro frontend with Islands architecture
- Plugin system with Hook mechanism
- Theme engine with hot-reload
- RESTful API with auto-generated Swagger docs
- JWT authentication with refresh tokens
- Two-factor authentication (TOTP)
- Role-based access control (RBAC)
- Article management system
- Comment system
- Media management with optimization
- Full-text search (Meilisearch integration)
- SEO optimization toolkit
- Docker deployment support
- Nginx reverse proxy configuration
- Supervisor process management
- Python SDK
- CLI management tools
- Database migrations with Alembic
- Redis caching layer
- Rate limiting
- CORS configuration
- Health check endpoints
- Audit logging
- Backup and restore functionality
- Email notification system
- Sitemap generation
- RSS feed support
- Multi-language support framework
